Общая информация
Программа "Система складского учёта ASKA" предназначена для учёта товаров на складе, учёта движения товара, для работы с удалёнными сервисными центрами.

Программа состоит из:
  • серверной части - где хранятся товары, документы, где реализована логика программы. FTP и HTTP сервер
  • клиентской части - набора форм и кнопок для доступа и управления информацией.
  • и клиентской программы для удалённых сервисных центров - исполняемый файл который позволяет создавать отчёты и заявки и отправлять их на сервер предприятия.

    Серверная часть программы построена на базе СУБД Cache компании Intersystems. Клиентская часть выполнена на web технологии и управляется посредством Internet Explorer. В комплекс входит также исполняемый файл RPTSender.exe для удалённых сервисных центров.
