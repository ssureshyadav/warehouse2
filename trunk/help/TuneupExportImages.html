Экспорт изображений
Данная функция позволяет подготовить все изображения товаров и запчастей либо для просмотра обычными программами для изображений, либо для приложения "Менеджер отчётов".
Первое что спросит программа - это каталог куда будут экспортированы файлы изображений. По умолчанию это c:\Goods Необходимо иметь ввиду, что это каталог на сервере где установлена Сервисная база. Далее необходимо ответить на вопрос - в каком виде должны быть экспортированы изображения: либо для Менеджера отчётов, либо для просмотра любым редактором изображений. Разница в том, что в первом случае все файлы будут размещены в одном каталоге и называться будут по номеру запчасти или товара, которому принадлежат. А в случае экспорта для просмотра - все изображения будут размещены в каталогах аналогично товарным группам и называться будут по названию товара или запчасти.
