Основные требования к системе компьютерам работающим с программой СКЛАД

На сервере: (обязательно) FTP Сервер
В корне диска С: должен лежать файл архиватор 7z.exe
(необязательно)
Для еженочного сохранения базы используется backup.cmd (см в каталоге с CSP файлами).

На клиентском компьютере
Должен быть установлен Internet Explorer версии не ниже 6.0.2800
IP адрес сервера должен быть занесён в безопасную зону.
Должен стоять Microsoft Office
